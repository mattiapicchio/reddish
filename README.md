## Reddish

## Setup

Install the Node version listed in [`.nvmrc`](.nvmrc).

```sh
nvm use
```

### Development server

Start the development server on [http://localhost:3000](http://localhost:3000)

```bash
npm run dev
```

### Production server
Build the application for production:

```sh
npm run build
```

Locally preview production build:

```sh
npm run start
```
